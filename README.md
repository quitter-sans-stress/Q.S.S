# QuitterSansStress.fr

**QuitterSansStress** est un service en ligne qui aide les salariés à quitter leur emploi sans stress, sans confrontation directe avec l’employeur, tout en respectant la loi française.

## ✨ Fonctionnalités principales

- Connexion sécurisée par email
- Tableau de bord personnel
- Aide humaine via WhatsApp ou email
- Génération automatique de lettre de démission (à venir)

## ⚙️ Stack technique

- Next.js
- NextAuth (authentification email)
- Vercel (hébergement)
